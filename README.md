# 📊 Sentiment Analysis Web App

A simple **Machine Learning web application** that predicts the **sentiment of text** as **Positive, Negative, Neutral, or Irrelevant**.
The application is built using **Python, Machine Learning, and Streamlit** to provide an interactive user interface.

---

## 🚀 Features

* 🔍 Predicts sentiment of user-entered text
* 😊 Detects **Positive sentiment**
* 😡 Detects **Negative sentiment**
* 😐 Detects **Neutral sentiment**
* ❓ Detects **Irrelevant sentiment**
* ⚡ Fast predictions using a trained machine learning model
* 🌐 Interactive web interface using **Streamlit**

---


## 💻 Usage

1. Enter any text in the input box.
2. Click the **Predict Sentiment** button.
3. The model will analyze the text and display the sentiment.

Example:

| Text                   | Predicted Sentiment |
| ---------------------- | ------------------- |
| "I love this product!" | Positive            |
| "This is terrible."    | Negative            |
| "It is okay."          | Neutral             |

## 🧠 Machine Learning Model

The model uses **Natural Language Processing (NLP)** techniques including:

* Text preprocessing
* Tokenization
* TF-IDF Vectorization
* Machine Learning Classification

The trained model predicts sentiment categories based on the input text.
